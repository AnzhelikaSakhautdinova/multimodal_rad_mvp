# Multimodal Video RAG Telegram Bot

Этот проект реализует простую мультимодальную систему поиска видео по текстовому запросу.

Пользователь отправляет текстовый запрос в тг-бот, а система находит и возвращает наиболее релевантные кадры и формирует короткое видео из библиотеки.

## Идея

Видео разбивается на кадры с временными метками (кадры разбивали не через сегментацию, а просто по временному интервалу).

Для каждого кадра или сцены вручную создаётся краткое текстовое описание.

Описания преобразуются в эмбеддинги с помощью модели Sentence Transformers и индексируются в FAISS.

При поступлении запроса бот находит наиболее релевантные сцены и возвращает соответствующие фрагменты видео.

## Принцип работы

```text
Видео
→ Извлечение кадров
→ Ручное описание сцен
→ Создание текстовых эмбеддингов
→ Индексация в FAISS
→ Telegram-бот
→ Поиск Top-k релевантных видеофрагментов
```


##  Пример запроса

```text
Запрос пользователя:

family

Ответ бота:

Запрос: family

1. 00:54 | 0.74
a family gathered in a kitchen...

2. 01:18 | 0.68
a family walking outdoors...

3. 01:42 | 0.61
a father and child doing exercises...
```

Число рядом с временной меткой отражает степень семантического сходства между запросом пользователя и описанием сцены.

## Установка

```text
Клонировать репозиторий:
git clone https://github.com/AnzhelikaSakhautdinova/multimodal_rad_mvp

Установить зависимости:
pip install -r requirements.txt

Построение индекса:
python build_index.py

Запуск Telegram-бота:
python bot.py
```
